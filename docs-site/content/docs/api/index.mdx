# API Reference

Welcome to the HORUS API reference documentation. This section provides detailed documentation for all public types, traits, and functions in the HORUS framework.

## Crates

| Crate | Description |
|-------|-------------|
| [horus_core](/api/core) | Core runtime - nodes, communication, scheduling |
| [horus_messages](/api/messages) | Standard message types for robotics |
| [horus_macros](/api/macros) | Procedural macros (`node!`, `message!`) |

---

## Quick Reference

### Core Types

| Type | Description | Module |
|------|-------------|--------|
| [`Node`](/api/core#node) | Base trait for all computation units | `horus_core` |
| [`Hub<T>`](/api/core#hub) | Multi-producer multi-consumer pub/sub channel | `horus_core` |
| [`Link<T>`](/api/core#link) | Single-producer single-consumer channel | `horus_core` |
| [`Scheduler`](/api/core#scheduler) | Node execution orchestrator | `horus_core` |
| [`NodeInfo`](/api/core#nodeinfo) | Runtime context for nodes | `horus_core` |

### Error Handling

| Type | Description | Module |
|------|-------------|--------|
| [`HorusError`](/api/core#horuserror) | Unified error type | `horus_core` |
| [`HorusResult<T>`](/api/core#horusresult) | Result alias | `horus_core` |

| Type | Description | Module |
|------|-------------|--------|

## Import Patterns

### Minimal Import

```rust
use horus::prelude::*;
```

### Full Import

```rust
use horus_core::{
    // Core traits and types
    Node, NodeInfo, NodeConfig, NodeMetrics, NodeState,
    // Communication
    Hub, Link, LinkMetrics,
    // Scheduling
    Scheduler,
    // Errors
    HorusError, HorusResult,
    // Traits
    Channel, Publisher, Subscriber,
};

use horus_messages::{
    Image, LaserScan, Imu, Twist, Pose, PointCloud,
};
```
